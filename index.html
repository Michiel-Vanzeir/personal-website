Line follower

Using a Raspberry Pi 3B+ and a camera, this robot can follow a black line or be controlled by a simplistic Flutter app. The software is written using ROS, C++ and Python. The OpenCV library is used to detect the middle of the line, which a PID controller uses to control the robot.
