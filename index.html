Programming Portfolio

WALL-E

WALL-E is a simple differential drive robot. Working on this project, I've learned both ROS & ROS2, control loop mechanisms like a PID and Fuzzy Logic controller, and Gazebo simulations. WALL-E can follow a black line using its camera videofeed and a PID controller, or be controlled with a keyboard.

WALL-E (simulated)

This repository contains the files needed to simulate WALL-E using Gazebo and integrating ROS with it. At the moment, only a URDF file representing the robot can be found in the repository. I'm currently working on simulating SLAM and using a Kalman filter to merge raw IMU sensor data.
